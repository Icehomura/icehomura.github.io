VI/VIM使用速查

发表于2023-10-04|更新于2023-10-08|实用工具
|阅读量:

本文参考整理了下面两个项目，感谢项目作者的付出

https://github.com/skywind3000/awesome-cheatsheets

快速指南

全局命令

命令 描述
:help 关键词 打开关键词的帮助
:o 文件 打开文件
:saveas 文件名 另存为
:close 关闭当前窗格

光标移动

命令 描述
h 向左移动光标
j 向下移动光标
k 向上移动光标
l 向右移动光标
H 移动到屏幕顶部
M 移动到屏幕中间
L 移动到屏幕底部
w 跳转到下一个单词的开头
W 跳转到下一个单词的开头（单词可以包含标点符号）
e 跳转到下一个单词的结尾
E 跳转到下一个单词的结尾（单词可以包含标点符号）
b 跳转到上一个单词的开头
B 跳转到上一个单词的开头（单词可以包含标点符号）
0 跳转到行的开头
^ 跳转到行的第一个非空字符
$ 跳转到行的末尾
g_ 跳转到行的最后一个非空字符
gg 跳转到文档的第一行
G 跳转到文档的最后一行
5G 跳转到第5行
fx 跳转到下一个字符x的位置
tx 跳转到字符x的前一个位置
} 跳转到下一段落（或函数/块，当编辑代码时）
{ 跳转到上一段落（或函数/块，当编辑代码时）
zz 将光标居中于屏幕
Ctrl + b 向后滚动一个完整屏幕
Ctrl + f 向前滚动一个完整屏幕
Ctrl + d 向前滚动半个屏幕
Ctrl + u 向后滚动半个屏幕

插入模式 - 插入/追加文本

命令 描述
i 在光标前插入
I 在行的开头插入
a 在光标后插入（追加）
A 在行的末尾插入（追加）
o 在当前行下方追加（打开）新行
O 在当前行上方追加（打开）新行
ea 在单词的末尾插入（追加）
Esc 退出插入模式

编辑

命令 描述
r 替换一个字符
J 将下一行连接到当前行
cc 更改（替换）整行
cw 更改（替换）到下一个单词的开头
ce 更改（替换）到下一个单词的结尾
cb 更改（替换）到上一个单词的开头
c0 更改（替换）到行的开头
c$ 更改（替换）到行的末尾
s 删除字符并替换文本
S 删除行并替换文本（与cc相同）
xp 交换两个字母（删除并粘贴）
. 重复上一条命令
u 撤销
Ctrl + r 重做

标记文本（可视模式）

命令 描述
v 开始可视模式，标记行，然后执行命令（如y-yank）
V 开始行可视模式
o 移动到标记区域的另一端
O 移动到标记区域的另一个角落
aw 标记一个单词
ab 标记带有()的块
aB 标记带有{}的块
ib 标记带有()的内部块
iB 标记带有{}的内部块
Esc 退出可视模式
Ctrl + v 开始可视块模式

可视命令

命令 描述
> 向右移动文本
< 向左移动文本
y 复制标记的文本
d 删除标记的文本
~ 切换大小写

剪切和粘贴

命令 描述
yy 复制一行
2yy 复制2行
yw 复制从光标位置到下一个单词开头的字符
y$ 复制到行末
p 在光标后粘贴剪贴板内容
P 在光标前粘贴剪贴板内容
dd 删除一行
2dd 删除2行
dw 删除从光标位置到下一个单词开头的字符
D 删除到行末
d$ 删除到行末
d^ 删除到行的第一个非空字符
d0 删除到行的开头
x 删除一个字符

搜索和替换

命令 描述
/pattern 搜索模式
?pattern 向后搜索模式
\vpattern very magic 模式：非字母数字字符被解释为特殊的正则表达式符号（无需转义）
n 在同一方向上重复搜索
N 在相反方向上重复搜索
:%s/old/new/g 在整个文件中替换所有的old为new
:%s/old/new/gc 在整个文件中替换所有的old为new，并进行确认
:noh 取消搜索匹配的高亮显示

在多个文件中搜索

命令 描述
:vimgrep /pattern/ {file} 在多个文件中搜索模式
:cn 跳转到下一个匹配项
:cp 跳转到上一个匹配项
:copen 打开包含匹配项列表的窗口

退出

命令 描述
:w 写入（保存）文件，但不退出
:w !sudo tee % 使用sudo保存当前文件
:wq or :x or ZZ 写入（保存）并退出
:q 退出（如果有未保存的更改则失败）
:q! or ZQ 退出并放弃未保存的更改

处理多个文件

命令 描述
:e 文件 在新缓冲区中编辑文件
:bnext or :bn 转到下一个缓冲区
:bprev or :bp 转到上一个缓冲区
:bd 删除缓冲区（关闭文件）
:ls 列出所有打开的缓冲区
:sp 文件 在新缓冲区中打开文件并分割窗口
:vsp 文件 在新缓冲区中打开文件并垂直分割窗口
Ctrl + ws 分割窗口
Ctrl + ww 切换窗口
Ctrl + wq 关闭窗口
Ctrl + wv 垂直分割窗口
Ctrl + wh 将光标移动到左侧窗口（垂直分割）
Ctrl + wl 将光标移动到右侧窗口（垂直分割）
Ctrl + wj 将光标移动到下方窗口（水平分割）
Ctrl + wk 将光标移动到上方窗口（水平分割）

标签页

命令 描述
:tabnew or :tabnew 文件 在新标签页中打开文件
Ctrl + wT 将当前分割窗口移动到新标签页
gt or :tabnext or :tabn 移动到下一个标签页
gT or :tabprev or :tabp 移动到上一个标签页
<number>gt 移动到标签页
:tabmove <number> 将当前标签页移动到第个位置（从0开始索引）
:tabclose or :tabc 关闭当前标签页及其所有窗口
:tabonly or :tabo 关闭除当前标签页外的所有标签页
:tabdo command 在所有标签页上运行命令（例如：:tabdo q 关闭所有打开的标签页）

详细指南

光标移动

命令 描述
h 光标左移，同 <Left> 键
j 光标下移，同 <Down> 键
k 光标上移，同 <Up> 键
l 光标右移，同 <Right> 键
Ctrl + F 下一页
Ctrl + B 上一页
Ctrl + U 上移半屏
Ctrl + D 下移半屏
0 跳到行首（是数字零，不是字母O），效用等同于 <Home> 键
^ 跳到从行首开始第一个非空白字符
$ 跳到行尾，效用等同于 <End> 键
gg 跳到第一行，效用等同于 Ctrl+<Home>
G 跳到最后一行，效用等同于 Ctrl+<End>
nG 跳到第n行，比如 10G 是移动到第十行
:n 跳到第n行，比如 :10<回车> 是移动到第十行
10% 移动到文件 10% 处
15| 移动到当前行的 15 列
w 跳到下一个单词开头 (word: 标点或空格分隔的单词)
W 跳到下一个单词开头 (WORD: 空格分隔的单词)
e 跳到下一个单词尾部 (word: 标点或空格分隔的单词)
E 跳到下一个单词尾部 (WORD: 空格分隔的单词)
b 上一个单词头 (word: 标点或空格分隔的单词)
B 上一个单词头 (WORD: 空格分隔的单词)
ge 上一个单词尾
) 向前移动一个句子（句号分隔）
( 向后移动一个句子（句号分隔）
} 向前移动一个段落（空行分隔）
{ 向后移动一个段落（空行分隔）
<enter> 移动到下一行首个非空字符
+ 移动到下一行首个非空字符（同回车键）
- 移动到上一行首个非空字符
H 移动到屏幕上部
M 移动到屏幕中部
L 移动到屏幕下部
fx 跳转到下一个为 x 的字符，2f/ 可以找到第二个斜杆
Fx 跳转到上一个为 x 的字符
tx 跳转到下一个为 x 的字符前
Tx 跳转到上一个为 x 的字符前
; 跳到下一个 f/t 搜索的结果
, 跳到上一个 f/t 搜索的结果
<S-Left> 按住 SHIFT 按左键，向左移动一个单词
<S-Right> 按住 SHIFT 按右键，向右移动一个单词
<S-Up> 按住 SHIFT 按上键，向上翻页
<S-Down> 按住 SHIFT 按下键，向下翻页
gm 移动到行中
gj 光标下移一行（忽略自动换行）
gk 光标上移一行（忽略自动换行）

插入模式

进入退出

命令 描述
i 在光标处进入插入模式
I 在行首进入插入模式
a 在光标后进入插入模式

命令 描述
<Up> 光标向上移动
<Down> 光标向下移动

<Right> 光标向右移动
<S-Left> 按住 SHIFT 按左键，向左移动一个单词
<S-Right> 按住 SHIFT 按右键，向右移动一个单词
<S-Up> 按住 SHIFT 按上键，向上翻页
<S-Down> 按住 SHIFT 按下键，向下翻页
<PageUp> 上翻页
<PageDown> 下翻页

<End> 光标跳转行尾
Ctrl + W 向前删除单词
Ctrl + O 临时退出插入模式，执行单条命令又返回插入模式
Ctrl + \ Ctrl + O 临时退出插入模式（光标保持），执行单条命令又返回插入模式

Ctrl + F 自动缩进

Ctrl + V {char} 插入非数字的字面量

Ctrl + V u1234 插入 16 进制 unicode 字符（四数字）
Ctrl + V U12345678 插入 16 进制 unicode 字符（八数字）
Ctrl + K {ch1} {ch2} 插入 digraph（见 :h digraph），快速输入日文或符号等
Ctrl + D 文字向前缩进
Ctrl + T 文字向后缩进

查找替换

命令 描述
/pattern 从光标处向文件尾搜索 pattern
?pattern 从光标处向文件头搜索 pattern
n 向同一方向执行上一次搜索
N 向相反方向执行上一次搜索
* 向前搜索光标下的单词
# 向后搜索光标下的单词
:s/p1/p2/g 将当前行中全替换p1为p2
:%s/p1/p2/g 将当前文件中全替换p1为p2
:%s/p1/p2/gc 将当前文件中全替换p1为p2，并且每处询问你是否替换
:10,20s/p1/p2/g 将第10到20行中所有p1替换为p2
:., ns/p1/p2/g 将当前行到n行中所有p1替换为p2
:., +10s/p1/p2/g 将当前行到相对当前行加10行的区间中所有p1替换为p2

小贴士

  • 永远不要用 Ctrl + C 代替 <ESC> 完全不同的含义，容易错误中断运行的后台脚本
  • 很多人使用 Ctrl + [ 代替 <ESC>，左手小指 Ctrl，右手小指 [ 熟练后很方便
  • 某些终端中使用 Vim 8 内嵌终端如看到奇怪字符，使用 :set t_RS= t_SH= 解决
  • 某些终端中使用 Vim 8.2+ 会看到一些奇怪字符，使用 :set t_TI= t_TE= 解决
  • 某些终端中使用 NeoVim 如看到奇怪字符，使用 :set guicursor= 解决
  • 使用 MS-Terminal 如果进入 Vim/NVim 会默认替换模式设置 :set t_u7= 解决
  • 多使用 ciw,ci[,ci", ci( 以及diw, di[, di", di( 命令来快速改写/删除文本
  • 在行内左右移动光标时，多使用w b e或W B E，而不是h l或方向键，这样会快很多
  • SHIFT 相当于移动加速键， w b e 移动光标很慢，但是 W B E 走的很快
  • 自己要善于总结新技巧，比如移动到行首非空字符时用 0w 命令比 ^ 命令更容易输入
  • 在空白行使用 dip 命令可以删除所有临近的空白行，viw 可以选择连续空白
  • 缩进时使用 >8j >} <ap >ap =i} == 会方便很多
  • 插入模式下，当你发现一个单词写错了，应该多用 Ctrl + W 这比 <BackSpace> 快
  • y d c 命令可以很好结合 f t 和 /X 比如 dt) 和 y/end<cr>
  • c d x 命令会自动填充寄存器 "1 到 "9 , y命令会自动填充 "0 寄存器
  • 用 v 命令选择文本时，可以用 o 掉头选择，有时很有用
  • 写文章时，可以写一段代码块，然后选中后执行 :!python 代码块就会被替换成结果
  • 搜索后经常使用 :nohl 来消除高亮，使用很频繁，可以 map 到 <BackSpace> 上
  • 搜索时可以用 Ctrl + R Ctrl + W 插入光标下的单词，命令模式也能这么用
  • 映射按键时，应该默认使用 noremap ，只有特别需要的时候使用 map
  • 当你觉得做某事很低效时，你应该停下来，u u u u 然后思考正确的高效方式来完成
  • 用y复制文本后，命令模式中 Ctrl + R 然后按双引号 0 可以插入之前复制内容
  • 某些情况下 Vim 绘制高亮慢，滚屏刷新慢可以试试 set re=1 使用老的正则引擎
  • Windows 下的 GVim 可以设置 set rop=type:directx,renmode:5 增强显示
